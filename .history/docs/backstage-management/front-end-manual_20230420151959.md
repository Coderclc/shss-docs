# 前端手册

## 前端组件

封装了一些常用的 JS 组件方法。

| 名称     | 代码        | 介绍             |
| -------- | ----------- | ---------------- |
| 表格     | $.table     | 表格封装处理     |
| 表格树   | $.treeTable | 表格树封装处理   |
| 表单     | $.form      | 表单封装处理     |
| 弹出层   | $.modal     | 弹出层封装处理   |
| 操作     | $.operate   | 操作封装处理     |
| 校验     | $.validate  | 校验封装处理     |
| 树插件   | $.tree      | 树插件封装处理   |
| 通用方法 | $.common    | 通用方法封装处理 |

- 表格初始化方法 `$.table.init`
